
NAME
    C:\Users\Tobias\git\world-office\scripts\Sync-Remotes.ps1
    
SYNOPSIS
    Sync git remotes for Euro-Office workspace repositories.
    
    
SYNTAX
    C:\Users\Tobias\git\world-office\scripts\Sync-Remotes.ps1 [-DryRun] [-Execute] [-AllowDirty] [-AutoAddRemotes] 
    [-AutoOurs] [-Force] [-Json] [[-RepoRoot] <String>] [-Help] [-ListRepos] [<CommonParameters>]
    
    
DESCRIPTION
    This script synchronizes git repositories in the Euro-Office workspace by:
    - Fetching from both upstream (world-office) and source (world-office) remotes
    - Merging changes from source remote into local branches
    - Pushing updates to world-office remote (Codeberg)
    
    By default, operates in dry-run mode to preview actions before execution.
